# Customer Segmentation Project

This project focuses on segmenting customers of a UK-based online gift retailer using RFM analysis and K-Means clustering.

## Project Structure
- **/notebooks**: Jupyter Notebook containing data cleaning, EDA, and clustering models.
- **/reports**: Final project report in PDF format.
- **/presentation**: Slide deck for the project presentation.

## Key Insights
- Successfully segmented customers into distinct groups (Champions, At-Risk, etc.) to improve marketing ROI.
- Validated segments using the Silhouette Score (0.594).

## Requirements
To run the analysis, install the necessary libraries:
`pip install -r requirements.txt`
